# lab-5
El repositorio contiene los archivos correspondientes al laboratorio #5 del curso de Programación Bajo Plataformas Abiertas (IE-0117).

Se implementó una lista enlazada así como las siguientes funciones: -Inserción de nodo al inicio. 
1. Inserción de nodo al final. 
2. Inserción de nodo en una posición específica. 
3. Búsqueda de un nodo a través de su dato. 
4. Eliminación de un nodo a través de su dato. 
5. Impresión de la lista hacia adelante.

Pasos para compilar: 
1. Clone el repositorio localmente. 
2. Ejecute en la terminal "make" 
3. Para eliminar los archivos objeto puede ejecutar "make clean".

Pasos para ejecutar: 
1. Ejecute en la terminal "./main"
